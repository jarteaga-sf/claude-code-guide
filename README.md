# The SE AI Toolkit

An interactive guide to AI tools for Salesforce SEs — covering **Saleo**, **Cursor**, and **Claude Code**.

Built as a companion site for the "How I AI" enablement session. The first three sections are designed for live virtual presentation (slide carousels), and the rest serves as a take-home reference with tool deep-dives, workflows, and a cheat sheet.

Hosted at **[https://jarteaga-sf.github.io/se-ai-toolkit/](https://jarteaga-sf.github.io/se-ai-toolkit/)** — just open the link.

![Screenshot](public/screenshot.png)

## What's Inside

| Tier | Sections | Format |
|------|----------|--------|
| **The Big Picture** | The Shift, The Tools, Get Set Up | Slide carousels (presentation mode) |
| **The Tools** | Claude Code, Cursor, Saleo | Tabbed deep-dives (reference) |
| **See It in Action** | Live Demo, The Full Pipeline | Walkthroughs |
| **Keep Going** | Cheat Sheet | Quick reference |

## Tech Stack

- Vite + React
- Tailwind CSS v4
- shadcn/ui components
- Lucide React icons

## Getting Started

```bash
npm install
npm run dev        # localhost:5173/se-ai-toolkit/
npm run build      # production build
```
